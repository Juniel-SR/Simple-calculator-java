Calculadora em Java

Uma calculadora simples desenvolvida em Java capaz de realizar operações aritméticas básicas.
Sobre o Projeto

Este projeto foi criado com o objetivo de praticar lógica de programação e conceitos de Programação Orientada a Objetos (POO) utilizando Java.
A calculadora realiza as seguintes operações:

Soma
Subtração
Multiplicação
Divisão

Também conta com interação com o usuário e tratamento básico de entradas.

Tecnologias Utilizadas
Java
JOptionPane (interação com o usuário)
